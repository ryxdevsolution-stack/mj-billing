## 4. Billing System

The billing system is the core of RYX Billing. It supports both GST and Non-GST billing.

### 4.1 GST Billing

**[SCREENSHOT: GST Billing Page]**

GST billing is used when you need to issue tax invoices with GST breakdown.

#### When to Use GST Billing:
- B2B (Business to Business) transactions
- When customer needs a tax invoice
- Transactions above a certain amount
- When customer provides GSTIN

#### Creating a GST Bill - Step by Step:

**Step 1: Open Billing Page**
- Click on **Billing** in the sidebar
- Select **GST Billing** or **Create New Bill**

**Step 2: Add Customer Details**

| Field | Description |
|-------|-------------|
| Customer Name | Name of the customer |
| Phone Number | Contact number |
| GSTIN | Customer's GST number (for B2B) |
| Address | Billing address |

**[SCREENSHOT: Customer Details Form]**

**Step 3: Add Products**

**[SCREENSHOT: Product Selection]**

- Search product by **name** or **barcode**
- Enter **quantity**
- Product rate and GST are automatically applied
- Click **Add** to add to bill

**Step 4: Review Bill**

The bill shows:
- Item-wise breakdown
- Subtotal
- GST Amount (calculated automatically based on product GST %)
- Discount (if applied)
- Final Amount

**Step 5: Apply Discount (Optional)**

| Discount Type | Description |
|--------------|-------------|
| Percentage | Apply X% discount on total |
| Flat Amount | Reduce fixed amount from total |

**Step 6: Select Payment Method**

- Cash
- UPI
- Card
- Credit (Due payment)

**Step 7: Save & Print**

- Click **Save Bill** to save
- Bill is automatically saved and stock is reduced
- Print receipt if needed

#### GST Rates Explained:

| Rate | Typically Used For |
|------|-------------------|
| 5% | Essential items, food products |
| 12% | Processed food, electronics |
| 18% | Most goods and services |
| 28% | Luxury items, automobiles |

### 4.2 Non-GST Billing

**[SCREENSHOT: Non-GST Bill]**

Non-GST billing is used for simple sales without tax breakdown.

#### When to Use Non-GST Billing:
- Small cash transactions
- When GST is not applicable
- Quick sales to walk-in customers
- Internal transactions

#### Creating a Non-GST Bill:

The process is same as GST billing, but:
- No GST calculation
- No GSTIN field
- Simpler receipt format

### 4.3 Managing Bills

**[SCREENSHOT: Bills List]**

#### Viewing All Bills:
- Go to **Billing** > **All Bills**
- See list of all bills with:
  - Bill Number
  - Customer Name
  - Date
  - Amount
  - Payment Status

#### Searching Bills:
- Search by bill number
- Search by customer name
- Filter by date range
- Filter by payment type

#### Editing a Bill:
- Open the bill you want to edit
- Click **Edit**
- Make changes
- Save

> **Note:** Editing may be restricted based on your user role.

#### Cancelling a Bill:
- Open the bill
- Click **Cancel Bill**
- Provide reason (optional)
- Confirm cancellation

> **Important:** Cancelled bills are not deleted but marked as cancelled for audit purposes.

#### Draft Bills:
- If a bill is not completed, it is saved as a draft
- Draft bills can be resumed later
- System shows notification for pending drafts

**[SCREENSHOT: Draft Bill Notification]**

### 4.4 Bill Exchange / Returns

**[SCREENSHOT: Exchange Page]**

When a customer wants to exchange or return products:

1. Go to **Billing** > **Exchange**
2. Find the original bill
3. Select items to return
4. Add new items (if exchanging)
5. Calculate difference
6. Complete the exchange

## 5. Inventory Management

The inventory module helps you track all your products and stock levels.

### 5.1 Stock Overview

**[SCREENSHOT: Inventory Dashboard]**

The inventory page shows:

- Total stock value
- Low stock items

### 5.2 Adding New Products

**[SCREENSHOT: Add Product Form]**

To add a new product, fill in the following:

| Field | Description | Required |
|-------|-------------|----------|
| Product Name | Name of the product | Yes |
| Category | Product category (e.g., Electronics, Grocery) | Yes |
| Cost Price | Your purchase price | Yes |
| Selling Rate | Price you sell at | Yes |
| MRP | Maximum Retail Price | Optional |
| Quantity | Current stock quantity | Yes |
| Unit | Unit of measurement (Pcs, Kg, Ltr, etc.) | Yes |
| GST Percentage | GST rate (5%, 12%, 18%, 28%) | For GST products |
| HSN Code | Harmonized System Nomenclature code | For GST products |
| Barcode | Product barcode number | Optional |
| Item Code | Your internal product code | Optional |
| Low Stock Alert | Quantity at which to show alert | Optional |

### 5.3 Managing Stock

**[SCREENSHOT: Stock Update]**

#### Adding Stock (When new stock arrives):
1. Find the product
2. Click **Add Stock**
3. Enter quantity to add
4. Stock is automatically increased

#### Stock Adjustment:
- For corrections or damaged goods
- Enter adjustment reason
- Stock is updated

#### Automatic Stock Reduction:
- When you create a bill, stock is automatically reduced
- No manual intervention needed
- Prevents overselling

### 5.4 Bulk Operations

#### Importing Products from Excel:

#### Bulk Stock Orders:
- Create orders for multiple products at once

### 5.5 Low Stock Alerts

**[SCREENSHOT: Low Stock Alert]**

RYX Billing supports multiple payment methods:

### 9.2 UPI Payment

### 9.3 Card Payment
- Credit/Debit card payments
- Can add last 4 digits for reference

### 9.4 Credit / Due Payment
- Customer pays later
- Bill is saved with "Due" status

- Mark as paid when received

### 9.5 Partial Payment
- Customer pays part of the amount
- Remaining is marked as due
- Track amount received vs pending

## 15. Glossary

| Term | Meaning |
|------|---------|
| **GST** | Goods and Services Tax - India's indirect tax |
| **GSTIN** | GST Identification Number - Unique 15-digit tax ID |
| **HSN Code** | Harmonized System Nomenclature - Product classification code |
| **MRP** | Maximum Retail Price - Maximum price at which product can be sold |
| **Bill Number** | Unique identifier for each bill |
| **SKU** | Stock Keeping Unit - Unique product identifier |
| **Barcode** | Machine-readable product code |
| **Thermal Printer** | Printer that uses heat to print on special paper |
| **POS** | Point of Sale - Where transactions happen |
| **Audit Trail** | Record of all actions for accountability |
| **Credit Sale** | Sale where payment is received later |
| **Stock Entry** | Record of product and its quantity |
